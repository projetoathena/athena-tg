Sobre

Este aplicativo tem por objetivo ser um sistema multiplataforma e responsivo, denominado Athena, para auxiliar no estudo da Teoria dos Grafos. Esse sistema permite a criação de grafos, edição de propriedades básicas de vértices e arestas, além da visualização de algumas propriedades básicas dos grafos. Por meio dele é possível, também, realizar a execução de alguns algoritmos passo a passo, permitindo que a visualização do desenvolvimento do algoritmo seja transparente para o estudante. Na modelagem do problema foram consideradas questões de usabilidade e experiência do usuário, para que o sistema fosse elaborado de forma a ser o mais amigável possível para o usuário iniciante nos estudos de Teoria dos Grafos. No desenvolvimento deste trabalho foram feitas pesquisas sobre outras soluções similares. O Athena foi modelado em Unified Modeling Language (UML) e desenvolvido usando-se as tecnologias JavaScript, HyperText Markup Language (HTML) e Cascading Style Sheets (CSS), alem do angular como framework e das bibliotecas de js, como:
